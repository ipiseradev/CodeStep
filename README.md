# 🚀 CodeStep

**CodeStep** es una plataforma de aprendizaje de programación enfocada en la ejecución real.

No es un curso.
No es teoría.
Es práctica constante.

> 💡 Aprendé programando todos los días con desafíos cortos, feedback inmediato y progreso medible.

---

## 🧠 ¿Qué es CodeStep?

CodeStep está diseñado como un sistema de aprendizaje tipo:

* 🎮 Duolingo → progreso gamificado
* 💻 LeetCode → desafíos prácticos
* ⚡ Mentalidad atleta → disciplina diaria

El objetivo es simple:

👉 que el usuario programe todos los días
👉 que vea progreso real
👉 que se mantenga constante

---

## ⚙️ ¿Cómo funciona?

Cada sesión está basada en **misiones cortas (2–5 minutos)**:

1. Recibís un desafío
2. Escribís o elegís la solución
3. El sistema valida automáticamente
4. Recibís feedback inmediato
5. Ganás XP
6. Desbloqueás el siguiente nivel

---

## 🧩 Sistema de aprendizaje

Los contenidos están organizados en tracks:

* 🟢 Sintaxis & Variables
* 🟡 Lógica condicional
* 🔵 Bucles & repetición
* 🟣 Programación modular

Cada módulo:

* se desbloquea progresivamente
* tiene validación automática
* contribuye a tu progreso global
